# 💳 Pricing Table UI

Uma interface moderna de tabela de preços desenvolvida com foco em apresentação de planos, comparação de recursos e conversão de usuários, utilizando um layout profissional para destacar diferentes opções de assinatura de forma clara e intuitiva.

---

## 📸 Preview do Projeto

Interface contendo:

- Três planos de assinatura (Basic, Extended e Premium)
- Destaque visual para preços
- Comparação de recursos entre planos
- Indicadores de funcionalidades disponíveis
- Botões de ação para compra
- Layout moderno e responsivo

---

## ✨ Tecnologias Utilizadas

- **HTML5**
- **CSS3**

---

## 🎨 Características do Design

- Interface moderna e minimalista
- Cartões de preços organizados em colunas
- Paleta de cores vibrante e atrativa
- Destaque visual para valores dos planos
- Comparação intuitiva de recursos
- Botões de ação personalizados
- Efeitos hover suaves
- Layout responsivo
- Design limpo e profissional

---

## 🚀 Objetivo do Projeto

Praticar habilidades de desenvolvimento front-end através da criação de uma tabela de preços moderna e responsiva, explorando conceitos de layout em colunas, hierarquia visual, componentes reutilizáveis e estilização avançada com CSS.

---

## 📚 Funcionalidades

- Exibição de múltiplos planos de assinatura
- Comparação visual de recursos
- Destaque para preços promocionais
- Indicadores de funcionalidades inclusas
- Botões de compra para cada plano
- Estrutura adaptável para diferentes dispositivos
- Fácil personalização de planos e valores
